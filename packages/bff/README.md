# BFF (Backend for Frontend)

Express.js API server with TSOA, Prisma, and tsyringe DI.

## Architecture

```
src/
├── config/        # Environment validation (envalid)
├── container.ts   # tsyringe DI registration
├── controllers/   # TSOA controllers (define routes + OpenAPI spec)
├── services/      # Business logic + Prisma access
│   └── __tests__/ # Vitest unit tests
└── generated/     # Auto-generated by TSOA (gitignored)
```

## Adding a New Endpoint

1. **Create a service** in `src/services/my-feature.service.ts`
2. **Create a controller** in `src/controllers/my-feature.controller.ts`
3. **Run** `pnpm tsoa:spec-and-routes` to regenerate routes + OpenAPI spec
4. **Regenerate the api-client** from the root: `pnpm generate:client`

### Controller Pattern

```typescript
import { Controller, Get, Route, Tags } from 'tsoa';
import { injectable } from 'tsyringe';
import type { MyDto } from 'common';

@Route('my-feature')
@Tags('MyFeature')
export class MyFeatureController extends Controller {
  @Get('/')
  public async getAll(): Promise<MyDto[]> {
    return [];
  }
}
```

## Prisma

### Adding a Model

1. Edit `prisma/schema.prisma`
2. Run `pnpm prisma:migrate` (creates + applies migration)
3. Prisma Client is auto-regenerated

### Migration Workflow

```bash
# Create and apply a migration
pnpm prisma:migrate

# Apply existing migrations (production)
pnpm exec prisma migrate deploy

# Open Prisma Studio
pnpm prisma:studio
```

## Environment Variables

| Variable         | Required | Default                  | Description                     |
|-----------------|----------|--------------------------|----------------------------------|
| `DATABASE_URL`  | ✅       | —                        | PostgreSQL connection string     |
| `NODE_ENV`      | ❌       | `development`            | `development`, `production`, `test` |
| `PORT`          | ❌       | `3000`                   | Server port                      |
| `HOST`          | ❌       | `0.0.0.0`               | Server host                      |
| `CORS_ORIGIN`   | ❌       | `http://localhost:5173` | Allowed CORS origin              |
| `ENABLE_SWAGGER`| ❌       | `true`                   | Enable Swagger UI at `/api-docs` |

## Testing

```bash
# Run all tests
pnpm test

# Run with coverage
pnpm test:coverage
```

### Test Pattern

Tests live in `src/services/__tests__/`. Use Vitest globals (`describe`, `it`, `expect`).
Mock external dependencies (Prisma, env) using `vi.mock()`.

```typescript
import { describe, it, expect, vi } from 'vitest';

describe('MyService', () => {
  it('does something', () => {
    expect(true).toBe(true);
  });
});
```
